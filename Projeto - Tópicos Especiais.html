TDD

TDD significa Desenvolvimento Orientado por Testes (Test Driven Development), e trata-se de uma prática de desenvolvimento de software onde a codificação das funcionalidades começa a partir da escrita de testes unitários. Essa técnica foi criada por Kent Beck e é um dos pilares do XP (Extreme Programming).

FLASK

Lançado em 2010, o Flask é um micro-framework que tem como objetivo desenvolver com facilidade aplicativos web. Sendo assim, o flask oferece simplicidade e rapidez para projetos pequenos e até mesmo apps maiores. Ferramentas do flask: WSGI utilizada no desenvolvimento com Python; Werkzeug compreendida com um kit de implementação de funções; Jinja2 refere-se a um sistema de templates que através de uma fonte de dados específica para deixar uma página renderizada.
